# Bold Questions Style

When you ask the user a question — whether for clarification, confirmation, or offering choices — you MUST format the question in **bold** text so it stands out clearly.

## Rules

1. Any sentence that is a direct question to the user (ends with `?`) MUST be wrapped in `**...**` bold markdown.
2. If you are presenting a list of choices/options, bold the leading question before the list.
3. This applies to all questions: clarifying questions, confirmation prompts, option selections, and follow-ups.
4. Non-question text (explanations, code, insights, etc.) should NOT be bolded unless it is already appropriate to bold.

## Examples

Good:
- **Which approach would you prefer?**
- **Do you want me to run the tests?**
- **Should I use the existing config or create a new one?**

Bad (don't do this):
- Which approach would you prefer?
- Do you want me to run the tests?
